# jiraline
Command Line Inteface for JIRA

----

# Settings

In file `~/.jiraline` put the following contents:


```
{
    "domain" : <your JIRA domain @ attlasian.net e.g. posbit>,
    "credentials":{
        "user":<your user name>,
        "password":<your password>
     }
}
```

Sample valid config:


```
{
    "domain" : "posbit",
    "credentials":{
        "user":"example"
        "password":"secret_password1"
     }
}
```
