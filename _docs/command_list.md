**assign**

Syntax:

```
jiraline assign -[u] <ISSUE_ID>
```

Options:

* u (-- user name). After option you have to type user name to who issue will be assigned to

**comment**

Syntax:

```
jiraline comment -[m] <ISSUE_ID>
```

Options:

* m - message. After option you have to type text of the message
